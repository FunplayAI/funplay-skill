---
description: "Choose the right engine-facing workflow before editing project files"
---

Tell the user you are using `using-funplay-skills` to route the task to the correct engine workflow.

Then:

1. Ask which engine or project type is in scope if it is not already obvious.
2. Use `unity-prefab-workflow` for Unity scenes, prefabs, materials, and serialized assets.
3. Use `godot-scene-assembly` for Godot scenes, nodes, resources, and exported properties.
4. Use `cocos-component-workflow` for Cocos Creator prefabs, components, and asset references.
