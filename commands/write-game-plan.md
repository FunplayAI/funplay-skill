---
description: "Turn an approved game concept into implementation tasks"
---

Tell the user you are using the `gameplay-prototyping` skill to produce a concrete build plan.

Then:

1. Restate the approved concept and success criteria.
2. Break the work into small implementation tasks grouped by gameplay, content, UI, and polish.
3. Include asset-processing steps when `sprite-sheet`, `normal-map`, or `audio-format-convert` are relevant.
4. If engine-specific file edits are expected, also use one of:
   - `unity-prefab-workflow`
   - `godot-scene-assembly`
   - `cocos-component-workflow`
