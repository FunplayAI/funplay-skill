---
description: "Refine a rough game idea into a prototype-ready concept"
---

Tell the user you are using the `using-funplay-skills` and `gameplay-prototyping` skills.

Then:

1. Clarify the player fantasy, core loop, target platform, and art style.
2. Produce a compact prototype concept that is small enough to build quickly.
3. Explicitly call out what is in scope for the first playable and what is deferred.
4. If the user approves, recommend running `/write-game-plan`.
