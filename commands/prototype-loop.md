---
description: "Refine a gameplay loop into a small, testable prototype slice"
---

Tell the user you are using `gameplay-prototyping`.

Then:

1. Reduce the proposed game to one repeatable loop that can be tested quickly.
2. Identify the first success/failure state the player should reach.
3. List the minimum mechanics, content, UI, and assets required.
4. Explicitly defer anything decorative or non-essential.
